# Monster Puppet

Game co-development studio specializing in technical art and character animation. Founded in 2023 by industry veterans with an average of 15 years of AAA game development experience.

## What We Do

- **Rigging** — Production-ready character rigs built for animators
- **Tools** — DCC and in-engine pipeline tools
- **Pipeline** — Asset integration and cross-department workflows
- **Animation** — Mocap and keyframe, stylized to photorealistic
- **Engineering** — Gameplay, animation engineering, and custom engine tools

## Selected Projects

- The Legend of California — Technical Art
- SUPERVIVE — Rigging, Tools, Character Pipeline
- NBA The Run — Rigging, Pipeline, Technical Animation
- Chronicles: Medieval — Technical Art, Technical Animation
- GOALS — Rigging, Pipeline
- GRAFT — Technical Animation, Animation
